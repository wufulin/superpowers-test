# 宝丽来风格照片生成器 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位

一款在线 Web 应用，让用户能够轻松将普通照片转换为具有复古宝丽来胶片风格的艺术照片。

### 1.2 目标用户

- 摄影爱好者
- 社交媒体内容创作者
- 追求复古美学的年轻用户
- 需要特殊风格照片的设计师

### 1.3 核心价值

- 一键生成宝丽来风格照片
- 无需安装，浏览器即用
- Canvas前端驱动的高质量图像转换

## 2. 功能需求

### 2.1 首屏页（Landing Page）

**页面元素：**

- 产品名称和标语
- 视觉吸引力强的示例对比图（原图 vs 宝丽来风格）
- 主要 CTA 按钮："开始制作" / "立即体验"
- 简短的功能说明

**设计风格：**

- 现代简约，突出产品核心功能
- 使用复古色调和元素暗示宝丽来主题
- 响应式设计，适配各种屏幕尺寸

### 2.2 照片上传功能

**交互流程：**

1. 用户点击"开始制作"按钮
2. 进入上传界面
3. 支持两种上传方式：
   - 点击上传区域选择文件
   - 拖拽文件到上传区域
4. 显示上传进度
5. 上传完成后预览原图

**技术要求：**

- 支持常见图片格式：JPG, PNG, WEBP
- 文件大小限制：最大 10MB
- 前端图片预览
- 图片尺寸验证

### 2.3 图像处理与生成

**处理流程：**

1. 用户确认上传的图片
2. 点击"生成宝丽来风格"按钮
3. 显示处理中状态（加载动画）
4. 使用Canvas进行风格转换
5. 展示生成结果

**宝丽来风格特征：**

- 柔和的色调和对比度
- 略微泛黄的复古色彩
- 白色边框（上窄下宽的典型宝丽来相框）
- 轻微的颗粒感和纹理
- 柔和的光晕效果

### 2.4 结果展示与下载

**功能要素：**

- 大图预览生成的宝丽来风格照片
- 原图/效果图对比切换
- 下载按钮（保存为 JPG/PNG）
- "再做一张"按钮返回上传界面
- 社交分享选项（可选）

## 3. 用户交互流程 (User Flow)

```
[首屏页]
   ↓ (点击"开始制作")
[上传界面]
   ↓ (选择/拖拽照片)
[照片预览 + 确认]
   ↓ (点击"生成宝丽来风格")
[处理中动画]
   ↓ (AI 模型处理)
[结果展示页]
   ↓ (下载 / 再做一张)
[返回上传界面] 或 [结束]
```

## 4. 技术架构

### 4.1 前端技术栈

- **框架：** Next.js 16
- **UI 库：** React + TypeScript 5
- **样式：** Tailwind CSS
- **构建：** Next.js 16 静态导出 (Static Site Generation)

### 4.2 图像处理

- **纯前端处理：** 使用 HTML5 Canvas API
- **宝丽来滤镜算法：**
  - 色调调整（偏暖色、泛黄效果）
  - 对比度和饱和度降低
  - 添加颗粒噪点
  - 添加宝丽来白色边框（上部窄、下部宽）
  - 轻微模糊和光晕效果
- **优势：** 无需服务器，纯客户端处理，响应速度快

### 4.3 部署方案

- 静态网站部署（Next.js `output: 'export'`）

## 5. UI/UX 设计原则

### 5.1 视觉设计

- **色彩方案：** 奶油白、复古黄、柔和棕色、淡青色
- **字体：** 现代无衬线字体（主标题）+ 手写风格字体（装饰元素）
- **图标：** SVG 图标，简约现代风格
- **动画：** 微交互动画增强用户体验

### 5.2 交互设计

- 流畅的页面过渡效果
- 明确的状态反馈（上传中、处理中、完成）
- 大而明显的 CTA 按钮
- 拖拽上传的直观指引

### 5.3 响应式设计

- 优先移动端体验
- 平板和桌面端适配
- 触摸友好的交互元素

## 6. 性能要求

- 首屏加载时间 < 2 秒
- 图片上传响应即时
- AI 处理时间明确告知用户
- 优化图片大小以加快处理速度

## 7. 后续优化方向（V2.0）

- 提供多种胶片风格选择（富士、柯达等）
- 滤镜强度调节
- 批量处理功能
- 用户作品画廊
- 移动端 PWA 支持

## 8. 成功指标

- 用户完成率（首屏 → 下载）> 60%
- 平均处理时间 < 30 秒
- 用户满意度 > 4.0/5.0
